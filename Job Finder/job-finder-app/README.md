# 🌌 DigyNex Job Finder: Elite Career Engine (V18.6)
**Automate your job search with the power of Neural Intelligence.**

Welcome to the future of job hunting. DigyNex Job Finder is a high-fidelity, autonomous application engine that handles everything from job discovery to form submission, ensuring you stay ahead of the market with zero friction.

---

## 🚀 CORE CAPABILITIES
*   **Neural Auto-Apply**: Headless automation (Puppeteer + Browserless) that completes applications on your behalf.
*   **Adaptive Jitter**: Randomized interaction patterns to bypass anti-bot detection and mimic human behavior.
*   **Gemini-Vision Ready**: Advanced AI visual verification for 100% submission accuracy.
*   **Global Discovery**: Multi-source scraping (Adzuna, Careerjet) tailored to global regions.
*   **Quota Governance**: Fair, post-charge quota management synced with Supabase.

---

## 🛠️ THE TECHNOLOGY STACK
*   **Frontend**: Vue.js 3, Vite, Vanilla CSS.
*   **Backend**: n8n (Strategic Signal Routing), Supabase (Identity & Activity).
*   **Engine**: Puppeteer, Node.js, Browserless.io.
*   **Intelligence**: Google Gemini 1.5 Pro/Flash.

## ⚙️ QUICK START (FOR DEVELOPERS)
1.  **Clone & Install**:
    ```bash
    git clone https://github.com/digynex/digynex-official.git
    npm install
    ```
2.  **Environment Setup**:
    Add your Supabase credentials to `.env`.
3.  **Run Development**:
    ```bash
    npm run dev
    ```
4.  **Deployment**:
    Use the `docker-compose.yml` to spin up the n8n automation stack on your Linux server.
