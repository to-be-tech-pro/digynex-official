## 🌍 1. The Core Vision
**DigyNex 360** is not just a dashboard; it is a **Single Source of Truth** for enterprise management. It bridges the gap between fragmented data (emails, spreadsheets, legacy tools) and executive decision-making. High-speed, secure, and AI-augmented.

## 🏗️ 2. The Architectural Pillar (Layered Intelligence)
*   **High-Fidelity Presentation Layer (Frontend):** Vue 3 + Tailwind CSS. A ultra-responsive, premium interface designed for high-density information display.
*   **Cloud-Native Data Core (Backend):** Supabase (PostgreSQL). Real-time synchronization with recursive-protected data isolation.
*   **The Automation Engine (n8n):** The "Central Nervous System" that connects internal UI actions to external alerts (WhatsApp, Telegram, Email).
*   **AI Insight Layer (Gemini):** Strategic pattern recognition for revenue forecasting and automated expense scanning (v2.0).

## 🧩 3. Module Deep-Dive (Module-by-Module Breakdown)

### A. 📊 BI Dashboard (The Strategic Nerve Center)
- **Who it's for:** CEO & Upper Management.
- **What it does:** Aggregates real-time KPIs (Revenue, Net Profit, Growth Targets).
- **Key Feature:** "Q1 Growth Target" tracker with dynamic drill-down analytics for precise performance review.

### B. ⚙️ Active Operations (PO to Profit Pipeline)
- **Who it's for:** Operations Managers & Staff.
- **Workflow:** Purchase Order (PO) → Work Order (WO) → Invoice Generation.
- **The "Profit Logic":** Once a Work Order is marked as "Completed," the system **automatically** logs production expenses in the financial ledger to update Net Profit metrics in real-time.
- **Document Vault:** Securely store and index external PO/WO assets (PDFs/Images) for compliance.

### C. 🤝 Strategic CRM & Partner Program
- **Who it's for:** Sales Teams & Strategic Ambassadors.
- **Client Management:** High-density directory for managing client lifetimes.
- **Partner Program:** A specialized engine for Referral Partners. Tracks commissions (5-10%), referral counts, and automated lead attribution.
- **Public Portal:** External partners can register via `digynex.se`, which triggers an automated review workflow.

### D. 📜 Financial Profit Engine (The Ledger)
- **Who it's for:** CFOs & Accountants.
- **What it does:** Records every transaction (Income/Expense) with absolute precision.
- **Metrics:** Day-by-Day Net Profit calculations, P&L summaries, and automated ROI tracking for subcontractor work.

### E. 👥 Personnel Identity Hub (Security Gate)
- **Who it's for:** Executive Board (CEO/Manager only).
- **What it does:** Manages organizational identities (Staff, Subcontractors, Admins).
- **Security:** Strict **RBAC (Role-Based Access Control)**. No one except the CEO/Manager can modify personnel records or sensitive roles.

## 🏷️ 4. Universal Labeling System (ULS v2.0)
The system is built to adapt instantly. With a single configuration change, the entire UI terminologies (Headers, Buttons, Tables) shift to match the client's industry:
- **Project Mode:** "Clients", "Milestones", "Projects".
- **SaaS Mode:** "Accounts", "Usage", "Subscriptions".
- **Education Mode:** "Students", "Tuition Hub", "Academy".

---

## 🛡️ 5. Governance & Zero Trust Security
- **Data Isolation:** All organizations are separated by **Row Level Security (RLS)**. One client's data is never visible to another.
- **Audit Logging:** Every critical change (Invoice edit, Partner registration) is logged in the `activity_logs` table for enterprise-level traceability.
- **Encryption:** All sensitive credentials (API Keys, Secrets) are stored in the n8n Secure Credential Store or Docker environment variables.

## ⚡ 6. Automation & Outreach (n8n Connectivity)
- **WhatsApp Sync:** Real-time alerts to the CEO's phone for "New Partner Registration" or "High-Value PO Ingestion".
- **Telegram Guardian:** Instant security notifications if an unauthorized data change attempt is detected.
- **Lead Ingestion:** Web forms are processed asynchronously, ensuring the database is updated even while the CEO is offline.
