6. **Daily Work Reporting**: once a task or session is complete, provide a clear, concise report of what was done in a format that can be saved for future reference.
7. **Git Integration**: After completing the work correctly and verifying it, ALWAYS ask for permission to `git push`. Only proceed if I say "OK".
